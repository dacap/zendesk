Solved: Loading cursor flickering in Word 2010 and 2007

A customer experienced in Microsoft Word, the cursor rapidly flickering between the loading blue circle icon and the arrow cursor. Typing a document is disconcerting when the cursor is flashing, especially  when selecting text. Thankfully I was able to solve this issue for my customer.

Affected software

  • Microsoft Office 2010 Home & Business, Microsoft Office 2010 Home & Student
  • Microsoft Office 2007 Home & Business, Microsoft Office 2010 Home & Student

Cause

Either one or both of the following:

  • Default printer has driver incompatible with Microsoft Word
  • Print spooler service failed

Solution

  • Uninstall printer driver
  • Download latest printer driver from support or driver download section of manufacturer website, e.g. Canon printer? go to http://www.canon.com.au
  • Install printer driver
