How to: Add images to an existing Nextgen Gallery

The following applies to the NextGEN gallery plugin for Wordpress sites.

Add images to an existing Nextgen Gallery

  1. Sign into to your Wordpress dashboard.
  2. In the side bar, click Pages.
  3. Click the page that has the gallery.
  4. Click on the gallery element.
  5. The nextgen gallery editor will open, click Add Gallery / Images.
  6. Under Upload Images heading, select existing Gallery from drop down menu.
  7. Click Add files button.
  8. In the file upload window, select files to upload then click Open.
  9. Click Upload button.
  10. Once upload is complete, you can now make other changes to the gallery. Remember to click save button.

For further help, please consult the documentation for the Nextgen Gallery or post a question to our free community support forum.
